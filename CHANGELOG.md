# 📝 更新日志 (Changelog)

## [1.1.0] - 2026-03-09

### ✨ 新增功能
- **时间范围切换**：支持在侧边栏切换14日和30日统计周期
- **动态列名匹配**：自动根据选择的时间范围匹配对应的Excel列名
- **实时规则显示**：侧边栏实时显示当前选择的统计规则

### 🎨 界面优化
- 在侧边栏添加Radio按钮组件，支持一键切换时间范围
- 统计规则说明动态更新，清晰显示当前统计周期
- 改进错误提示信息，明确指出缺少的列名

### 🔧 技术改进
- 重构 `process_excel_file()` 函数，增加 `days` 参数
- 重构 `validate_file()` 函数，增加 `days` 参数
- 使用 f-string 动态生成列名，提高代码灵活性
- 保持向后兼容，默认值为14日

### 📝 文档更新
- 更新 README.md，增加30日统计说明
- 新增 CHANGELOG.md 记录版本历史
- 完善常见问题解答

## [1.0.0] - 2026-01-09

### 🎉 首次发布

#### ✨ 核心功能
- 批量上传Excel文件（最多10个）
- 自动提取批次号（从文件名）
- 14日数据统计：
  - 登录用户数统计
  - 消费用户数统计
  - 总消费金额计算
  - 499元套餐购买次数统计

#### 📊 数据展示
- 汇总表格展示（按批次号排序）
- 三组可视化图表：
  - 登录数 vs 消费数对比图
  - 总消费金额趋势图
  - 499元套餐购买数量图
- 总计数据展示

#### 💾 导出功能
- 一键下载Excel报告
- 包含完整统计结果
- 文件名自动生成时间戳

#### 🎨 界面设计
- 马卡龙淡紫色主题
- 清新优雅的配色方案
- 响应式布局
- 友好的错误提示

#### 🔧 技术实现
- 基于 Streamlit 框架
- 使用 Pandas 进行数据处理
- Plotly 提供交互式图表
- OpenPyXL 处理Excel文件

## 版本说明

### 版本号格式
- **主版本号（Major）**: 不兼容的API修改
- **次版本号（Minor）**: 向下兼容的功能性新增
- **修订号（Patch）**: 向下兼容的问题修正

### 更新类型标识
- ✨ **新增功能** (Added)
- 🎨 **界面优化** (Changed)
- 🔧 **技术改进** (Fixed)
- 🗑️ **删除功能** (Removed)
- ⚠️ **已弃用** (Deprecated)
- 🔒 **安全修复** (Security)
- 📝 **文档更新** (Documentation)
