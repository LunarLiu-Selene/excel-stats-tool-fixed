# 📊 Excel数据统计工具

一个优雅的订单销售数据批量分析系统，采用马卡龙淡紫色主题设计。

## ✨ 功能特点

- 📁 **批量上传**：支持一次性上传10个Excel文件
- 🎯 **智能统计**：自动提取批次号并按规则统计数据
- 📊 **可视化展示**：美观的图表展示统计结果
- 💾 **一键导出**：生成包含所有统计结果的Excel报告
- 🎨 **优雅界面**：马卡龙淡紫色+白色+蓝色点缀的清新设计

## 📋 统计规则

系统会对每个Excel文件进行以下统计：

| 列名 | 统计内容 | 数据来源 |
|------|----------|----------|
| **批次号** | 从文件名提取（如"第1001批"） | 文件命名 |
| **b列** | 14日内登录用户数 | "14日内是否登录"列中值为1的数量 |
| **c列** | 14日内消费用户数 | "14日内是否消费"列中值为1的数量 |
| **d列** | 14日内总消费金额 | "14日内会员消费金额" + "14日内声音包消费金额" + "14日内通用会员消费金额" |
| **e列** | 499元套餐购买次数 | "14日内购买记录"中包含"两年年套餐499元"或"UNKNOWN499元"的次数 |

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

### 3. 使用步骤

1. **上传文件**：点击上传区域，选择10个Excel文件（支持批量选择）
2. **开始统计**：点击"🚀 开始统计"按钮
3. **查看结果**：
   - 查看汇总表格（按批次号排序）
   - 浏览可视化图表
   - 查看总计数据
4. **导出报告**：点击"📥 下载Excel报告"保存结果

## 📝 文件命名规范

Excel文件必须遵循以下命名规范：

```
第[批次号]批[其他描述].xlsx
```

示例：
- ✅ `第1001批订单数据.xlsx`
- ✅ `第1002批销售统计.xlsx`
- ❌ `订单数据.xlsx`（缺少批次号）

## 📊 Excel表格格式要求

表格必须包含以下列：

- `14日内是否登录`
- `14日内是否消费`
- `14日内会员消费金额`
- `14日内声音包消费金额`
- `14日内通用会员消费金额`
- `14日内购买记录`

## 📄 输出说明

### 汇总表格

按批次号升序排列，包含以下列：
- 批次号
- b列(登录数)
- c列(消费数)
- d列(总金额)
- e列(套餐数)

### 可视化图表

1. **登录数 vs 消费数**：分组柱状图对比
2. **总消费金额趋势**：折线图展示金额变化
3. **499元套餐购买数量**：柱状图展示购买情况

### Excel报告

导出的Excel文件包含完整的统计结果表格，可直接用于报告或进一步分析。

## 🐛 常见问题

**Q: 上传文件后没有反应？**

A: 请确保文件命名包含"第xxx批"格式，且文件为.xlsx或.xls格式。

**Q: 统计结果为0？**

A: 请检查Excel表格中的列名是否与要求完全一致（区分大小写）。

**Q: 某些文件处理失败？**

A: 点击"查看错误详情"查看具体错误信息，通常是列名不匹配或数据格式问题。
