# 📊 Excel数据统计工具

一个优雅的订单销售数据批量分析系统，采用马卡龙淡紫色主题设计。

## ✨ 功能特点

- 📁 **批量上传**：支持一次性上传10个Excel文件
- 🔄 **灵活统计**：支持14日和30日两种统计周期切换
- 🎯 **智能统计**：自动提取批次号并按规则统计数据
- 📊 **可视化展示**：美观的图表展示统计结果
- 💾 **一键导出**：生成包含所有统计结果的Excel报告
- 🎨 **优雅界面**：马卡龙淡紫色+白色+蓝色点缀的清新设计

## 🆕 新功能（v1.1.0）

### 📅 时间范围切换

现在支持在侧边栏切换统计周期：
- **14日统计**：统计最近14日内的数据
- **30日统计**：统计最近30日内的数据

只需点击侧边栏的单选按钮即可切换，统计规则会自动更新！

## 📋 统计规则

### 14日统计

系统会对每个Excel文件进行以下统计：

| 列名 | 统计内容 | 数据来源 |
|------|----------|----------|
| **批次号** | 从文件名提取（如"第1001批"） | 文件命名 |
| **b列** | 14日内登录用户数 | "14日内是否登录"列中值为1的数量 |
| **c列** | 14日内消费用户数 | "14日内是否消费"列中值为1的数量 |
| **d列** | 14日内总消费金额 | "14日内会员消费金额" + "14日内声音包消费金额" + "14日内通用会员消费金额" |
| **e列** | 499元套餐购买次数 | "14日内购买记录"中包含"两年年套餐499元"或"UNKNOWN499元"的次数 |

### 30日统计

当切换到30日模式时，统计规则相同，但数据来源列名变为：

| 列名 | 统计内容 | 数据来源 |
|------|----------|----------|
| **批次号** | 从文件名提取（如"第1001批"） | 文件命名 |
| **b列** | 30日内登录用户数 | "30日内是否登录"列中值为1的数量 |
| **c列** | 30日内消费用户数 | "30日内是否消费"列中值为1的数量 |
| **d列** | 30日内总消费金额 | "30日内会员消费金额" + "30日内声音包消费金额" + "30日内通用会员消费金额" |
| **e列** | 499元套餐购买次数 | "30日内购买记录"中包含"两年年套餐499元"或"UNKNOWN499元"的次数 |

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

### 3. 使用步骤

1. **选择时间范围**：在侧边栏选择"14日"或"30日"统计周期
2. **上传文件**：点击上传区域，选择Excel文件（支持批量选择）
3. **开始统计**：点击"🚀 开始统计"按钮
4. **查看结果**：
   - 查看汇总表格（按批次号排序）
   - 浏览可视化图表
   - 查看总计数据
5. **导出报告**：点击"📥 下载Excel报告"保存结果

## 📝 文件命名规范

Excel文件必须遵循以下命名规范：

```
第[批次号]批[其他描述].xlsx
```

示例：
- ✅ `第1001批订单数据.xlsx`
- ✅ `第1002批销售统计.xlsx`
- ❌ `订单数据.xlsx`（缺少批次号）

## 📊 Excel表格格式要求

### 14日统计模式

表格必须包含以下列：
- `14日内是否登录`
- `14日内是否消费`
- `14日内会员消费金额`
- `14日内声音包消费金额`
- `14日内通用会员消费金额`
- `14日内购买记录`

### 30日统计模式

表格必须包含以下列：
- `30日内是否登录`
- `30日内是否消费`
- `30日内会员消费金额`
- `30日内声音包消费金额`
- `30日内通用会员消费金额`
- `30日内购买记录`

> 💡 **提示**：如果您的Excel文件同时包含14日和30日的列，可以随时切换统计周期进行对比分析！

## 🐛 常见问题

**Q: 上传文件后没有反应？**

A: 请确保文件命名包含"第xxx批"格式，且文件为.xlsx或.xls格式。

**Q: 统计结果为0？**

A: 请检查Excel表格中的列名是否与要求完全一致（区分大小写）。

**Q: 某些文件处理失败？**

A: 点击"查看错误详情"查看具体错误信息，通常是列名不匹配或数据格式问题。

**Q: 切换到30日后提示缺少列？**

A: 请确保您的Excel文件包含30日相关的列（如"30日内是否登录"等）。如果只有14日的列，请使用14日统计模式。

**Q: 如何对比14日和30日的数据？**

A：
1. 先选择"14日"，统计并下载报告
2. 再选择"30日"，统计并下载报告
3．对比两份Excel报告进行分析
